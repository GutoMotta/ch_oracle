# Choracle
Choracle é um projeto sendo desenvolvido em Python com uso da biblioteca [LibROSA](https://github.com/librosa/librosa) que tem como objetivo fazer reconhecimento automatizado de acordes em gravações de áudio.

A base teórica para os primeiros passos foi o capítulo 5 do livro [Fundamentals of Music Processing](http://www.springer.com/us/book/9783319219448).
